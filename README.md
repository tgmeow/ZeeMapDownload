# ZeeMapDownload
Download ZeeMap data to your disk as CSV

To quick commands to get this running on Linux:
```bash
sudo apt update
sudo apt install -y git nodejs npm
git clone https://github.com/tgmeow/ZeeMapDownload
cd ZeeMapDownload
npm install
cp headers.csv output.csv
npm run zmaps &

```
